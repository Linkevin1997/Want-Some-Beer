# WANT-SOME-BEER

- This project uses HTML CSS and Javascript. We fetch the API from: https://api.punkapi.com/v2/beers


## Installation

Install my-project with npm

```bash
  git clone https://github.com/Linkevin1997/Want-Some-Beer.git 
  ls Want-some-Beer
  cd Want-some-Beer
  open index.html
```
